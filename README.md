## Getting Started

**Step 1:**

Download or clone this repo by using the link below:

```
https://github.com/Shanu1515/Tasker-app.git
```

**Step 2:**

Go to project root and execute the following command in console to get the required dependencies: 

```
flutter pub get 
```

**Step 3:**

Use the following command to run your Flutter app on a connected device or simulator.

```
flutter run
```

## Concepts

1. Provider for simple state management.
2. Stream builder for reactive data.
3. SQLite for persistence. 
4. Uses moor as an abstraction layer on top of SQFlite.
